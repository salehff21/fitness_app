## 📌 Overview

A simple and efficient **offline-first fitness tracking app** that allows users to:

* Track workouts with timers
* Log meals and nutrition
* Monitor daily water intake
* View progress and history

All data is stored locally using **SQLite**, ensuring speed and privacy.

---

## 🧰 Tech Stack

* **Language:** Kotlin
* **Architecture:** Activities + Fragments
* **UI:** RecyclerView + ViewBinding
* **Database:** `androidx.sqlite`
* **Build System:** Gradle (Android Gradle Plugin)
* **Testing:** JUnit, AndroidX Test, Espresso

---

## ✨ Features

* 🏃 Track workouts with built-in timer
* 🍽️ Log meals and nutrition
* 💧 Track daily water intake
* 📊 View history and basic reports
* 🔌 Ready for Health Connect integration
